## Mogas MOC 分析功能（下午探索）

### eWindow API ✅ 已接通
- **Base URL**: `https://api.platts.com/tradedata/v3`
- **端点**: `/ewindowdata`
- **认证**: 复用 Platts Bearer Token（无需 appkey）
- **成交筛选**: `order_state="consummated"`（不是 order_type="Trade"！）
- **市场名称**: `ASIA Mogas Physical`、`ASIA Mogas Swap`、`ASIA Crude Partial` 等
- **脚本**: `scripts/mogas-moc-ewindow.mjs`
- 关键字段：order_type(Bid/Offer), order_state(active/inactive/withdrawn/consummated), order_spread(T/F), buyer_mnemonic, seller_mnemonic, market_maker_mnemonic, deal_id
- spread 成交会生成3条记录：1条 spread leg + 2条 outright derived legs，outright VWAP 计算应用 order_spread=F 过滤
- 全部74个市场可查：`groupBy=market&field=max(order_date)`

## Mogas MOPS Strip & Daily Structure 方法论（2026-03-12 深度研究）

### 核心算法（来自 Platts Asia Refined Oil Products Methodology PDF）

**MOPS 92 RON (PRFEY00) Assessment:**
- = MOC 窗口内 outright swap 成交 VWAP（order_spread="F"，consummated）
- ⚠️ 不含 spread 价差交易拆分的 derived legs（eWindow 会出现，但 Heards 不记录）
- 今日：7笔/200kb，VWAP = **$117.57/b**（注：eWindow 含衍生腿为 $117.58，以 Heards 为准）

**Daily Structure（每日结构）:**
- = Bal Month / M1 spread ÷ 两个月中点之间的天数
- 今日：spread = $12.00（Platts 编辑评估），days = 25 → Daily Structure = **$0.48/b**
- 源自 KDC，不是自己能独立算出的精确值

**MOPS Strip (AAXEQ00):**
- = Bal Month swap − Daily Structure × (mid-window − mid-BalMonth)
- 今日：$129.60 − 0.48 × 13.5 = **$123.12/b**（KDC 原文）
- 装货窗口 = 今日+15 到今日+30 天 = Mar27~Apr11，mid-window = day 34.5（从3月1日算）
- mid-Bal Month = floor((今日 + 月末) / 2) = floor((12+31)/2) = day 21
- mid-M1 (Apr) = 月末 + floor((1 + 30) / 2) = 31 + 15 = day 46
- days between = 46 − 21 = 25 ✓

**何时切换 Bal/M1 → M1/M2:**
- 当 15天远期进入下月时切换（约每月16-17日）

### KDC 数据对比（今日）
- KDC 用 spread = $12.00（在 bid $10 / offer $23.28 之间，偏近 bid 侧约15%分位）
- 我们估算范围：0%=$122.18 ~ 100%=$128.29，KDC 落在约 $123.12（接近25%分位）
- **结论：Platts 通常评估偏 bid 侧，约 bid + 15-25% 的 bid/offer range**

## EIA 周报脚本 Bug 修复

### 问题
Tianshu 发现委内瑞拉进口显示 +232（暗示上周为0），不符合逻辑。

### 根因
`eia-weekly-report.mjs` 的 `fetchCountryImports()` 函数：
- `length = weeks * 15 = 30`（仅获取30条记录）
- EIA wimpc 端点每个周期约返回 19 个国家
- 30条 = 最新周期19条 + 上一周期仅11条
- 委内瑞拉(NUS-NVE)排在第14位，在上一周期的11条里被截掉
- 导致 Venezuela/沙特/伊拉克 的上周值显示为 0

### 实际数据（EIA直接查询）
- 2026-03-06: 委内瑞拉 232 kb/d
- 2026-02-27: 委内瑞拉 143 kb/d（不是0）
- 正确变化应为 +89，而非 +232

### 修复
将 `weeks * 15` → `weeks * 50`，确保能覆盖2个完整周期的所有国家。
已验证修复后输出正确：委内瑞拉 +89，沙特 +87，伊拉克 +155。

### 影响

- 下次 EIA cron 将使用正确版本

## mogas-mops-strip.mjs Bug 修复（18:29 SGT）

### 问题
脚本输出显示 "from outright Bal Month bid/offer"，说明 spread 数据（Bal Month/Apr26 bid/offer）没有被识别到。

### 根因
`getBalMonthInputs()` 用单次 `ewGet()` 拉全天数据，ASIA Mogas Swap 全天约 900+ 条记录，而 eWindow 分页上限 500 条。spread bid/offer 记录排在 500 条之后，被截断。

### 修复
将单次查询拆分为 5 个独立并发查询（按 order_type + order_spread + order_state 过滤）：
- Bid + outright active/inactive
- Offer + outright active/inactive
- Bid + spread active/inactive
- Offer + spread active/inactive
- spread consummated

修复后正确识别 spread bid $10 (EQUINOR) / offer $23.28 (GUNVORSG)。
