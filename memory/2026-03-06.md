```json

```

### 关键发现
- **ackReactionScope** 默认 "group-mentions"，改为 "all" 才能在非@场景触发
- **removeAckAfterReply** 默认 false，改为 true 才能让 👀 在回复后消失
- **🦞 不是 Telegram 支持的反应表情**，不能用于 react
- **Telegram topic 图标**只支持 custom emoji (Topics 贴纸包)，不支持普通 emoji

### Reaction 使用原则
- 有意义时才加（确认、庆祝、幽默），不刷屏
- Telegram 支持的反应表情有限：👍 ❤️ 🔥 🎉 👀 等
