# Dubai MOC Report 完善记录 (2026-03-10)

## 问题
- Cron 每次生成的格式不固定，AI 自由发挥导致漂移
- 3月10日报告与3月9日（完美模板）格式不一致

## 解决方案
1. **格式锁定**: 在 cron payload 和 MEMORY.md 中写入严格模板约束
2. **技能创建**: `skills/dubai-moc-report/SKILL.md` — 完整的数据获取流程和格式规范
3. **数据存档**: `reports/moc-daily/YYYY-MM-DD.json` — Platts heards API 仅保留当天数据
4. **Cron 重构**: payload 改为先读 SKILL.md 再执行，确保一致性

## 关键教训
- FluxOfficials 有时少计 declarations（3月10日报6个，实际8个），以 Platts 逐条为准
- Platts heards API 滚动窗口仅当天，次日数据消失，必须实时存档
- Cash Oman 有时不发布（如 Oman partials 无成交时）
- 格式修改建议可提出，但需 Tianshu 批准后才能执行
