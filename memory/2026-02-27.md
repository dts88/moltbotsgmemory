## Platts Structured Heards API ✅ 完整学习

**三个端点**:
1. `GET /structured-heards/v1/markets` - 市场列表及其字段
2. `GET /structured-heards/v1/metadata` - 字段定义与类型 (142个字段)
3. `GET /structured-heards/v1/data` - 实际交易数据

**可用市场** (已验证):
| 市场 | 记录数 | 特点 |
|------|--------|------|
| Americas crude oil | 19,885 | 最完整，有 volume/location |
| Asia crude oil | 935 | 亚洲原油交易 |
| Platts Carbon | 14,953 | 碳信用交易 |
| EMEA crude oil | 0 | 暂无数据 |

⚠️ **限制**: 仅原油+碳市场，**不含成品油** (汽油/柴油/航煤)

**原油字段**:
- `heard_type`: Trade/Bid/Offer/Indicative value
- `grade`: 油种 (WTI MEH, Mars, Basrah Medium, Muda Condensate, Ichthys Condensate...)
- `price`: 价差 (+0.95, -1.00, +2...)
- `pricing_basis`: 基准 (WTI, Dated Brent, Dubai, OSP...)
- `volume`: 货量 (仅美洲原油，如 30,000 bbl)
- `laycan`: 装期 (March, April...)
- `location`: 交割地 (Midland, Clovelly, South Korea...)

**碳市场字段**:
- `credit_type`: REDD+, Afforestation, Household devices, Biochar...
- `certification_and_standards`: VCS-CCB, Gold Standard, Puro.earth...
- `vintage`: 年份 (2020, 2021, 2022+...)
- `price`, `volume`

**脚本**: `scripts/platts-structured-heards.mjs`
```bash
markets   # 市场列表
metadata  # 字段定义
table <market> --type=Trade --limit=10  # 表格
heards <market> --days=7  # 详情
export <market>  # JSON
```

**元数据**: `reports/structured-heards-metadata.json`
