## Platts Structured Heards API 深度学习 (续 2/27)

### Token 管理改进
- **问题**: 手动更新 token 时用 `Date.now() + 3600` 计算过期时间不准确
- **解决**: 从 JWT payload 中解析 `exp` 字段获取真实过期时间
- **代码**: `Buffer.from(token.split('.')[1], 'base64')` → `JSON.parse()` → `exp * 1000`

### Buyer/Seller 字段确认
- 测试了 2026年1月至今的所有数据
- **Asia crude oil**: 0% 填充率
- **Americas crude oil**: 0% 填充率
- **结论**: 虽然 schema 定义了这些字段，但实际数据中从未填充

### Dubai/Murban 基准交易查询
- Dubai 基准: 34 条 (WTI Midland, WTL, Das Blend, CPC Blend, DFC, Condensate...)
- Murban 基准: 3 条 (仅 Das Blend, -65¢ ~ -80¢)
- 主要目的地: 韩国、台湾
- 贸易条款: FOB/DES/CFR

### 实际可用字段
| 字段 | 说明 |
|------|------|
| heard_type | Trade/Bid/Offer/Indicative value |
| grade | 油种 |
| price + min/max | 价差/价格区间 |
| price_qualifier_1/2/3 | floating, exact, Around, Low, Mid, High |
| pricing_basis | Dubai/Murban/Dated Brent/WTI/OSP |
| laycan | 装期 |
| incoterm | FOB/DES/CFR |
| location | 目的地 |
