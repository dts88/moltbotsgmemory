# 成品油月报 - 四段式报告模板

## 段落结构

### 第一段：炼油毛利总览

**开头固定**："截至[日期]，亚洲地区炼油毛利为[DBSCM00]美元/桶，……"

内容要点：
- 亚洲综合炼油毛利（DBSCM00），说明当日数值及变化原因
- 若月均与当日差异显著，点明月均水平（体现周期趋势）
- 西北欧（布伦特基准）、美湾（WTI基准）与亚洲的对比，可简洁描述，无需具体数字（若无Platts官方评估）

### 第二段：汽油市场

内容要点：
- 92 RON裂解价差（AAYED00）当前数值及近期走势
- 亚洲主要需求/供应驱动因素（如中国出口限制、季节性需求等）
- 石脑油可在此段或第三段简要提及

### 第三段：馏分油（航煤/柴油）及其他成品油

内容要点：
- 航煤裂解价差（AAHCL00）+ 绝对价
- 柴油裂解价差（AAHCE00）+ 绝对价
- 两者为炼厂核心利润支撑，对比危机前数据体现涨幅
- 可加入重要的供应事件（港口停运、攻击事件等），语言简洁
- 石脑油裂解价差（AAVUD00 vs 迪拜；NBSSM01 vs 布伦特），可补充说明原油品种价差的不对称冲击

### 第四段：宏观与驱动因素

内容要点：
- 市场核心驱动：地缘/需求/供应
- 政策面或机构行动（如IEA释储、政府出口限制等）
- 分化格局总结（哪些品种强/弱及原因）
- 可简要提及前景判断，但不做明确价格预测

## 定稿样例（2026年3月16日版）

截至3月16日，亚洲地区炼油毛利（迪拜基准复合裂解）为-6.00美元/桶，受周末美军袭击哈尔格岛后迪拜原油跳升至153美元/桶拖累，原油涨幅超越成品油，综合利润转负。值得注意的是，开战以来月均炼油毛利约达+9.7美元/桶，远高于战前均值（约4美元/桶），盈利中枢整体上移。西北欧和美湾地区因分别使用布伦特和WTI计价原料，较亚洲炼厂享有约45–50美元/桶的成本优势，综合毛利均维持正值。

汽油市场走势相对疲软。92 RON FOB新加坡裂解价差截至3月16日为17.59美元/桶，较危机初期3月5日的21.74美元/桶有所收窄，随迪拜原油突破150美元/桶持续承压。中国政府于3月初要求炼厂暂停成品油出口，印度、泰国跟进类似限制，亚洲出口供应大幅收缩，季节性夏季备货需求有望提供短期支撑。

馏分油是本轮成品油板块的最大亮点。航煤裂解价差截至3月16日为60.64美元/桶（绝对价201美元/桶），柴油裂解价差42.14美元/桶（绝对价191美元/桶），较危机前（2月26日分别约22美元/桶）涨幅均超90%，均处历史高位，为炼厂利润的核心支撑。3月14日美军袭击哈尔格岛后，伊朗随即攻击富查伊拉港致装卸作业全面暂停，中东及东南亚地区航煤供应骤紧，部分航线已报告燃油短缺。石脑油裂解价差（vs迪拜）同期仍为-1.64美元/桶，相对布伦特则达+15.63美元/桶，两者差距集中反映迪拜与布伦特原油价差扩大对亚洲炼厂原料成本的不对称冲击。

宏观层面，霍尔木兹海峡通行自2月底近乎归零，约2000万桶/日的全球原油及成品油转运受阻，中东产油国出口停滞叠加亚洲主要消费国出口限制，成品油市场呈现高度分化——馏分油价差维持极端水平，汽油和石脑油相对滞涨。IEA 3月11日宣布释放4亿桶战略储备，市场普遍认为实质缓解仍有赖于霍尔木兹通行恢复。

*数据来源：Platts API（新加坡官方评估）；NWE及USGC毛利为参考估算。截至2026-03-16。*
