# 市场周报技能

生成能源市场周报 Word 文档，覆盖原油、成品油、燃料油、LNG 及相关地缘政治动态。

## 周期定义

- **周期**: 周六 00:00 → 周五 23:59 (SGT)
- **文件命名**: `2026-WXX.md` (ISO 周数)
- **输出文件**: `市场周报 2026年X月X日.docx`

## 工作流程

| 阶段 | 时间 | 内容 |
|------|------|------|
| **素材收集** | 周六→周四 | Platts Monitor 自动抓取 + 用户转发 |
| **素材整理** | 持续 | 汇总到 `reports/weekly-material/2026-WXX.md` |
| **试生成** | 周五白天 | 基于现有素材生成草稿，供用户审阅 |
| **正式版** | 周五 Platts 收盘后 | 加入当周收盘价，生成最终 Word |
| **周期切换** | 周五生成后 | 创建下周素材文件，转移未使用素材 |

## 素材来源

1. **Platts Monitor** (自动): Heards + Insight News
2. **Platts API**: 收盘价格数据
3. **EIA 周报**: 美国库存、产量、炼厂开工率
4. **金十数据** (jin10.com): 实时财经快讯、大宗商品要闻
5. **用户转发**: TD3C 运费、投行观点、地缘政治
6. **网络搜索**: 补充验证（投行报告、石油公司动态、宏观数据）

### 信息处理原则
- **去重**: 同一事件从多源获取时，保留信息最完整的版本
- **合并优化**: 相关信息整合到同一条目，避免碎片化
- **筛选标准**: 与之前一致（本周实际发生、有具体数据、影响价格走势）

## 素材筛选原则

### ✅ 收录
- 本周实际发生的市场动态
- 即将发生的事件/检修
- 有具体数据支撑的信息
- 影响价格走势的因素

### ❌ 排除
- 对过去较长时间事件的复盘总结
- 无具体数据的泛泛而谈
- 与能源市场无关的内容
- MOC 流程提醒等程序性信息

## Word 文档格式规范

### 字体与排版
- **字体**: 等线 (Dengxian)
- **字号**: 20pt
- **对齐**: 左右对齐 (Justify)
- **段后**: 8pt
- **行距**: 双倍行距

### 页面设置
- **页边距**: 上下 1 英寸，左右 1.25 英寸

### 章节结构
```
标题: 市场周报 2026年X月X日

一、价格走势
  [价格表格]

二、市场因素
  1）EIA周报数据...
  2）OPEC+动态...
  3）地缘政治...
  ...

三、成品油
  [炼厂利润、库存等]

四、TD3C（中东至中国）
  [运费、市场动态]
```

### 序号格式
- 市场因素使用: `1）` `2）` `3）`...
- 不使用 `1.` 或 `(1)`

## 价格表格结构

| 列 | 内容 |
|----|------|
| 品种 | 迪拜首行、布伦特首行、WTI首行、新加坡92号汽油、新加坡10ppm柴油、新加坡航煤、新加坡380燃料油 |
| 价格 (美元) | 收盘价 |
| 日涨跌 (美元) | 与前一日差值 |
| 日涨跌 (%) | 百分比 |
| 周涨跌 (%) | 与上周五差值百分比 |
| 价格 (人民币/吨) | 换算值 |
| 本周走势 | ↗ / ↘ / → |

### 单位换算
- 原油: 1吨 ≈ 7.33桶
- 汽油: 1吨 ≈ 8.33桶
- 柴油: 1吨 ≈ 7.46桶
- 航煤: 1吨 ≈ 7.88桶
- 燃料油: 已是美元/吨

## 各章节写作规范

### 三、成品油
- **格式**: 一段话，紧凑叙述，不分段不分条
- **结构**: 按品种逐一列出裂解利润的变化值和绝对值
- **顺序**: 石脑油 → 汽油 → 航煤 → 柴油 → 燃料油(180/380/VLSFO) → LPG → 综合利润
- **模板**: "本周亚洲炼厂利润整体走强/走弱/分化。石脑油加工利润上涨/下跌X美元至X美元/桶（CFR日本 vs 迪拜），汽油利润上涨X美元至X美元/桶，航煤利润...柴油利润...燃料油利润...LPG方面...整体来看，迪拜综合裂解利润上涨/下跌X美元至X美元/桶。"
- **数据来源**: Platts裂解价差API (周四数据对比上周四)
- **附加信息**: 可在末尾补充重大政策/事件影响（如出口限制、检修等），但仍在同一段内

### 四、TD3C（中东至中国）
- **格式**: 按主题综合叙述，**不要逐日罗列**
- **开头**: 一句话点明本周WS费率/包干运费/趋势（如"TD3C VLCC运费升至WSxxx，约X万美元/船，折合X美元/桶，本周强势收官"）
- **主体段落**: 按主题组织（中东市场 → 大西洋市场 → 整体总结），将每日素材综合提炼
- **语言**: 全中文，英文术语翻译（on subs→暂定成交, lump-sum→包干运费, MEG→中东, WAF→西非, USG→美湾, lifted→确认, failed→落空, Suezmax→苏伊士型）
- **金额**: 统一"X美元"格式，不用"$X"
- **可选**: 重大地缘事件时可加"专题"分析段

### 全局写作规则
- **语言一致性**: 正文全中文，专有名词首次出现可加英文注释如"延布（Yanbu）"
- **金额格式**: "X美元/桶"、"X美元/吨"，不混用$符号
- **不加粗**: 正文不使用加粗
- **序号**: 市场因素用"1）2）3）"，不用"1." "(1)"
