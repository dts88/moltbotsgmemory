# 市场周报技能

生成能源市场周报 Word 文档，覆盖原油、成品油、燃料油、LNG 及相关地缘政治动态。

## 周期定义

- **周期**: 周六 00:00 → 周五 23:59 (SGT)
- **文件命名**: `2026-WXX.md` (ISO 周数)
- **输出文件**: `市场周报 2026年X月X日.docx`

## 工作流程

| 阶段 | 时间 | 内容 |
|------|------|------|
| **素材收集** | 周六→周四 | Platts Monitor 自动抓取 + 用户转发 |
| **素材整理** | 持续 | 汇总到 `reports/weekly-material/2026-WXX.md` |
| **试生成** | 周五白天 | 基于现有素材生成草稿，供用户审阅 |
| **正式版** | 周五 Platts 收盘后 | 加入当周收盘价，生成最终 Word |
| **周期切换** | 周五生成后 | 创建下周素材文件，转移未使用素材 |

## 素材来源

1. **Platts Monitor** (自动): Heards + Insight News
2. **Platts API**: 收盘价格数据
3. **EIA 周报**: 美国库存、产量、炼厂开工率
4. **用户转发**: TD3C 运费、投行观点、地缘政治
5. **网络搜索**: 补充验证

## 素材筛选原则

### ✅ 收录
- 本周实际发生的市场动态
- 即将发生的事件/检修
- 有具体数据支撑的信息
- 影响价格走势的因素

### ❌ 排除
- 对过去较长时间事件的复盘总结
- 无具体数据的泛泛而谈
- 与能源市场无关的内容
- MOC 流程提醒等程序性信息

## Word 文档格式规范

### 字体与排版
- **字体**: 等线 (Dengxian)
- **字号**: 20pt
- **对齐**: 左右对齐 (Justify)
- **段后**: 8pt
- **行距**: 双倍行距

### 页面设置
- **页边距**: 上下 1 英寸，左右 1.25 英寸

### 章节结构
```
标题: 市场周报 2026年X月X日

一、价格走势
  [价格表格]

二、市场因素
  1）EIA周报数据...
  2）OPEC+动态...
  3）地缘政治...
  ...

三、成品油
  [炼厂利润、库存等]

四、TD3C（中东至中国）
  [运费、市场动态]
```

### 序号格式
- 市场因素使用: `1）` `2）` `3）`...
- 不使用 `1.` 或 `(1)`

## 价格表格结构

| 列 | 内容 |
|----|------|
| 品种 | 迪拜首行、布伦特首行、WTI首行、新加坡92号汽油、新加坡10ppm柴油、新加坡航煤、新加坡380燃料油 |
| 价格 (美元) | 收盘价 |
| 日涨跌 (美元) | 与前一日差值 |
| 日涨跌 (%) | 百分比 |
| 周涨跌 (%) | 与上周五差值百分比 |
| 价格 (人民币/吨) | 换算值 |
| 本周走势 | ↗ / ↘ / → |

### 单位换算
- 原油: 1吨 ≈ 7.33桶
- 汽油: 1吨 ≈ 8.33桶
- 柴油: 1吨 ≈ 7.46桶
- 航煤: 1吨 ≈ 7.88桶
- 燃料油: 已是美元/吨

## 注意事项

1. **周期边界**: 严格按周六-周五划分，不要混入上周或下周的内容
2. **数据核实**: 价格数据必须来自 Platts API，不要估算
3. **时效性**: 素材按日期标注，便于筛选
4. **连贯性**: 生成时检查上周周报，避免重复或遗漏
5. **格式一致**: 严格遵守 Word 格式规范，不要随意调整
